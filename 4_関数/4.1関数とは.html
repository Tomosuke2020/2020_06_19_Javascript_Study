関数とは

関数とは、与えられた入力（パラメーター）に基づいてなんらかの処理を行い、
その結果を返す仕組みを関数といいます。
Javascriptはデフォルトでも多くの関数を提供しており、
アプリ開発者が自分で関数を定義することもできます。

このアプリ開発者が自分で関数を定義することを標準の関数と区別して、
「ユーザー定義関数」といいます。

ユーザーの定義できるものは4つある

・function命令
・Functionコントラクタ経由で定義する
・関数リテラル表現で定義する
・アロー関数で定義する

4.1.1 function命令で定義する

JavaScriptで関数を定義する場合、
function命令で定義することが最も基本的なアプローチです。

function命令の構文

function 関数名（引数, ...）{
...任意の処理...
return 戻り値; }

関数名をつける場合に気を付けること

・（単なる文字列ではなく）識別子の条件を満たす必要がある。
・「その関数がどのような処理を担っているのか」がすぐにわかるような名前をつける
・「showMessage」のように「動詞＋名詞」の形式で命名するのが一般的

引数は関数の挙動を決めるためのパラメータです。
呼び出しもとから指定された値を受け取るための変数を、カンマ区切りで指定します。
仮変数ともいい、関数の内部でのみ参照できます。

戻り値（返り値）は関数が処理の結果、最終的に呼び出しもとに返すための値のためです。

通常、関数の末尾にreturn命令を記述して指定します。

関数のイメージ

引数（関数が動作するための必要なパラメータ）→関数（予め決められた処理）→戻り値（関数による処理の結果）

getTraiangle関数
・引数として変数basse(底辺)
・height(高さ)を受け取り、
戻り値として三角形の面積を返す関数です。

定義済みの関数は、以下のようにして呼び出します。

関数名（[引数,...]）;

引数が存在しない場合にも、
関数の後方のまるかっこは省略できないので、
注意してください。

JavaScriptでは組み込みオブジェクトとして
Functionオブジェクトを用意しています。
このfunctionオブジェクトのコンストラクタを利用して定義することもできます。

構文 Functionコンストラクター

var 変数名 = new Function(引数 ,... ,関数の本体);

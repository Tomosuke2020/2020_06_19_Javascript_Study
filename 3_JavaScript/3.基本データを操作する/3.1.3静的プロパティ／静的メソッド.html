静的プロパティ／静的メソッド

ただし、プロパティやメソッドによっては例外的にインスタンスを生成せずに利用できるものがあります。
このようなプロパティ／メソッドのことを
静的プロパティ／静的メソッド、またはクラスプロパティ／クラスメソッドと呼びます。

静的プロパティ／静的メソッドの呼び出し

オブジェクト名.プロパティ名[= 設定値];
オブジェクト名.メソッド名([引数[,...]]);

これら静的メソッドは、
インスタンス変数から呼び出そうとするとエラーになるので、
注意してください。
なお、インスタンスを経由して呼び出すプロパティ／メソッドのことを
インスタンスプロパティ／インスタンスメソッドといいます。

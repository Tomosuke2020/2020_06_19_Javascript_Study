JavaScriptで利用可能な正規表現

正規表現で表された文字列パターンのことを正規表現パターンといいます。
以下では、アプリを開発する際によく利用する主な正規表現パターンです。

なお与えられた正規表現パターンが、ある文字列の中に含まれる場合、文字列が正規表現パターンにマッチするといいます。

URLで正規表現を表す正規表現パターンを読み解いてみましょう。

http(s)?://([\w-]+\.)+[\w-]+(/[w- ./?%&=]*)?

3.5.2RegExpオブジェクト

var 変数名 = mew RegExp('正規表現' , 'オプション'); var 変数名 = /正規表現/オプション;

正規表現リテラルでは、
スラッシュ（/）で括らなければいけない点に要注意です。
変数名 + オプションで正規表現の挙動を決めることができます。

正規表現の検索の基本

matchメソッドの構文

str.match(pattern) str：検索対象の文字列 pattern:正規表現

最終文にオプションを加えることで変更を加えることができます。

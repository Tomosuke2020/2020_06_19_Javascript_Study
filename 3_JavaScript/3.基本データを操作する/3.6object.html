両者の役割はよく似ていますが、以下のような違いがあります。

・valueOfメソッド・・・文字列以外の基本型の値が返されることを「期待して」使われる

3.6.4 不変オブジェクトを定義する

不変オブジェクトとは、
最初にインスタンスを生成した後、
一切の状態（値）を変更できないオブジェクトのことをいいます。
